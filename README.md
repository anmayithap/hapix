# Конфигурации Nix

Этот репозиторий содержит конфигурации Nix для управления системой и пользовательскими настройками на основе NixOS и macOS.

## Структура репозитория

```text
├── _internal/              # Внутренние инструменты и общие утилиты
├── custom/                 # Пользовательские конфигурации
├── home/                   # Конфигурации домашних директорий
├── modules/                # Системные модули и сервисы
├── hosts/                  # Хост-специфичные настройки
├── flake.nix               # Основной файл конфигурации Nix-flake
└── README.md               # Документация
```

## Возможности

- Поддержка NixOS и macOS (Darwin)
- Модульная архитектура для простоты настройки
- Интеграция с агентами секретов (agenix)
- Автоматизированные проверки кода (pre-commit hooks)

## Применение конфигурации

Для применения конфигурации на macOS:

```bash
darwin-rebuild switch --flake .#maple
```
